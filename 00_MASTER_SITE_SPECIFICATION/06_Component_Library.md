# Perla Toscana - Component Library

## Hero

Purpose:

- Communicate the property, location and booking action immediately.

Required elements:

- Strong image.
- H1.
- Short value proposition.
- Primary CTA.
- WhatsApp or secondary CTA.
- Key trust/benefit chips.

Rules:

- Use real property or location imagery.
- Do not place the main experience inside a decorative card.
- Ensure text remains readable on mobile.

## Gallery

Purpose:

- Build visual trust.

Features:

- Responsive grid.
- Lightbox.
- Category filters if enough photos exist.
- Descriptive alt text.

Categories:

- Bedrooms.
- Kitchen.
- Bathroom.
- Living areas.
- Parking.
- Details.

Do not include garden photos until the garden is ready.

## Feature Grid

Purpose:

- Show key services and amenities.

Typical items:

- Private parking.
- Fast Wi-Fi.
- Air conditioning.
- Smart TVs.
- Equipped kitchen.
- Dishwasher.
- Hydromassage shower.
- Linens and towels.

Use icons consistently.

## Reviews

Purpose:

- Reduce booking hesitation.

Elements:

- Review text.
- Guest name or initials if allowed.
- Source if relevant.
- Rating only if real and compliant.

## FAQ

Purpose:

- Answer objections and support SEO.

Features:

- Accordion or simple visible list.
- Structured data support.
- Contextual links to Book Direct or services.

## Map

Purpose:

- Explain location and nearby attractions.

Elements:

- Google Maps embed or static map.
- Directions button.
- Distance list.
- Links to Florence, Chianti, The Mall and Figline pages.

Use exact address only after owner approval.

## CTA

Purpose:

- Drive direct booking and WhatsApp contact.

Variants:

- Inline CTA.
- Section CTA.
- Sticky mobile CTA.
- Footer CTA.

Rules:

- Primary CTA must be visually consistent.
- Do not overload pages with too many competing actions.

## Availability Section

Purpose:

- Capture booking intent.

Elements:

- Dates.
- Guests.
- Contact details.
- Message.
- Submit button.
- WhatsApp fallback.

## Nearby Attractions

Purpose:

- Connect location to trip planning.

Typical items:

- Florence.
- Chianti.
- The Mall Firenze.
- Figline e Incisa Valdarno.
- Greve in Chianti.
- Siena.
- San Gimignano.

Each attraction card must link to a relevant page.

## Related Articles

Purpose:

- Strengthen topical authority and internal linking.

Rules:

- Show only related content.
- Prioritize articles supporting the current landing page.
- Do not show random blog posts.

## Booking Section

Purpose:

- Final conversion block.

Must include:

- Direct booking CTA.
- WhatsApp CTA.
- Trust signals.
- Short reminder of key benefits.
- Link to FAQ or contact.
